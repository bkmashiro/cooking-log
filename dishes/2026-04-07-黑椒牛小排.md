# 黑椒牛小排

**日期：** 2026-04-07
**食材：** 牛小排（Short Ribs）

---

## 做法

### 食材（2人份）

| 食材 | 用量 |
|------|------|
| 牛小排 | 500g |
| 黑胡椒碎（粗磨） | 1.5 茶匙 |
| 盐 | 1 茶匙 |
| 蒜 | 3 瓣（切末） |
| 洋葱 | 半个（切丝） |
| 黄油 | 20g |
| 生抽 | 1 汤匙 |
| 牛肉高汤（或清水） | 100ml |
| 食用油 | 适量 |
| 淀粉 | 1 茶匙（可选） |

### 步骤

1. 牛小排用厨房纸彻底擦干
2. 两面抹盐 1茶匙 + 黑椒碎 1茶匙，常温腌制 30分钟~1小时
3. 铸铁锅大火烧到冒烟，加少量油
4. 骨头朝上放入，不动，煎 3-4 分钟至底部深褐色焦壳
5. 翻面再煎 2-3 分钟，侧面竖起封边 1 分钟
6. 出锅静置 5 分钟
7. 原锅转中小火，下黄油，炒蒜末+洋葱丝约 2 分钟
8. 加黑椒碎 0.5茶匙，炒香后倒生抽+牛肉汤，煮沸
9. 按需用淀粉水收汁，淋在排上

---

## 注意要点

- ✅ 肉要**彻底擦干**，否则煎不出焦壳
- ✅ 锅要**足够热**再下肉
- ✅ 煎完必须**静置**，不要马上切
- ✅ 黑椒用**粗磨**，香气更好
- 🌡️ 五分熟约 60°C，七分熟约 68°C
