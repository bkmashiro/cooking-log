**奇尔沙治**: **黑椒牛小排详细教程** 🥩

---

**📦 食材（2人份）**

| 食材 | 用量 |
|------|------|
| 牛小排 | 500g |
| 黑胡椒碎 | 1.5 茶匙（粗磨更香） |
| 盐 | 1 茶匙 |
| 蒜 | 3 瓣（切末） |
| 洋葱 | 半个（切丝） |
| 黄油 | 20g |
| 生抽 | 1 汤匙 |
| 牛肉高汤（或清水） | 100ml |
| 食用油 | 适量 |
| 淀粉 | 1 茶匙（可选，使酱汁更浓稠） |

---

**⏱ 腌制（提前30分钟~1小时）**

1. 牛小排用厨房纸**彻底擦干**（关键！保证煎出焦壳）
2. 两面均匀抹上 **盐 1茶匙 + 黑椒碎 1茶匙**
3. 常温静置腌制，不要放冰箱

---

**🔥 煎排（约8-10分钟）**

1. 铸铁锅或厚底锅**大火烧到冒烟**，加少量油
2. 牛小排**骨头朝上**放入，**不要动**，煎 **3-4分钟** 至底部深褐色焦壳
3. 翻面再煎 **2-3分钟**
4. 侧面竖起来煎边 **1分钟**（封住肉汁）
5. 出锅**静置5分钟**再切（锁住肉汁）

> 🌡️ 喜欢几分熟参考：五分熟内部约60°C，七分熟约68°C

---

**🍶 黑椒汁（煎完在同一个锅里做）**

1. 转中小火，放入 **黄油20g**，融化后下 **蒜末 + 洋葱丝** 炒香（约2分钟）
2. 加 **黑椒碎0.5茶匙** 炒出香气
3. 倒入 **生抽1汤匙 + 牛肉汤100ml**，煮沸
4. 如需浓稠：淀粉1茶匙兑水调匀，慢慢倒入收汁
5. 尝味，按喜好补盐/黑椒

---

**🍽️ 摆盘**

- 牛小排切块或整块摆盘
- 黑椒汁淋在上面
- 配 **黄油土豆泥** 或 **烤西兰花** 超搭

---

**💡 关键Tips**

- ✅ 肉要**擦干**，湿肉煎不出焦壳
- ✅ 锅要**足够热**再放肉
- ✅ 煎完一定要**静置**，别急着切
- ✅ 黑椒要用**粗磨**，香气差很多

**统一表示方案：YAML frontmatter**

每个菜品文件顶部加结构化数据，人类写菜谱内容，机器只读 frontmatter：

```yaml
- { name: 牛肉, amount: 500, unit: g, scale: linear }
- { name: 蒜,   amount: 3,   unit: 瓣, scale: discrete }
- { name: 食用油, scale: to_taste }
```

**四种 scale 类型覆盖所有情况：**
| 类型 | 行为 | 适用 |
|------|------|------|
| `linear` | 线性缩放 | 大多数食材 |
| `discrete` | 取整到 ½ 步进 | 整颗蔬菜、蒜瓣 |
| `to_taste` | 始终显示适量 | 食用油、盐等调味 |
| `fixed` | 原值不变 | 烤盘、固定设备 |

**CI 自动工作流：**
- 每次 push `dishes/` 就触发 → 生成份量表 → bot 自动 commit 回去
- 生成的表格用 sentinel 注释包裹，幂等操作，可以反复安全运行
- 支持 `workflow_dispatch` 手动触发全量回填
